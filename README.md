# PowerShell Repository
![PowerShell](https://img.shields.io/badge/PowerShell-%235391FE.svg?style=for-the-badge&logo=powershell&logoColor=white)

This repository contains a collection of PowerShell commands and scripts for automating tasks, managing systems, and working with cloud services. Each script is organised by functionality, making it simple to navigate and find what you need.

## What is PowerShell?

PowerShell is an open-source, cross platform, object-oriented command-line shell and scripting language developed by Microsoft.
[Learn more about PowerShell here](https://learn.microsoft.com/en-us/powershell/)

## What's in this Repository

This repository contains a variety of commands, scripts and functions that I have found useful in my work with PowerShell. Examples include:

- Scripts for automating tasks and processes
- Functions for working with data and text
- Commands for managing and configuring systems and networks
